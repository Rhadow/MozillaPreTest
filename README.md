# AutoComplete Plugin

##How to Start

After cloning to your own computer, run:

```
$ cd MozillaPreTest
$ npm install
$ npm run dev
```

then visit [http://localhost:8080/](http://localhost:8080/).

or visit [http://rhadow.github.io/MozillaPreTest/](http://rhadow.github.io/MozillaPreTest/) or online demo.
